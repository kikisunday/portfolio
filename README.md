# 椎久智也 - フロントエンドエンジニア ポートフォリオサイト

## 📋 プロジェクト概要

椎久智也さんのエンジニアポートフォリオサイトです。5年以上のフロントエンド開発経験、技術スキル、プロジェクト実績を紹介する自己紹介サイトとなっています。

## 🎯 目的

- **個人ブランディング**: エンジニアとしてのスキルと経験をアピール
- **プロジェクト実績の可視化**: これまで携わったプロジェクトを詳細に紹介
- **お問い合わせ窓口**: 新しい仕事の機会やコラボレーションの受付

## ✨ 主な機能

### 実装済み機能

1. **ヒーローセクション**
   - フルスクリーンのインパクトある導入部
   - タイピングエフェクト（"Frontend Engineer"）
   - アニメーショングリッド背景
   - 統計表示（経験年数、プロジェクト数、チーム規模）
   - CTAボタン（プロジェクト閲覧、お問い合わせ）

2. **Aboutセクション**
   - 自己PR・強みの紹介
   - 4つの主要な特徴をカード形式で表示
     * タスク管理と効率化
     * 継続的な学習
     * チーム貢献と教育
     * コミュニケーション重視

3. **Skillsセクション**
   - カテゴリ別のスキル一覧（4カテゴリ）
     * 言語（JavaScript/TypeScript、HTML/CSS、Python、Java、Ruby）
     * フレームワーク（Vue.js/Nuxt.js、React、Angular、Express/Serverless、Rails）
     * インフラ・ツール（AWS、Azure DevOps、Git、Slack、Redmine）
     * データベース・OS（PostgreSQL/MySQL、SQL Server、Linux、Windows Server）
   - 各スキルの経験年数と習熟度を表示
   - プログレスバーによる視覚的な習熟度表現

4. **Projectsセクション**
   - 主要プロジェクト4件を詳細に紹介
     * 通信キャリアオンラインサイトの開発・保守（現在進行中）
     * インターネットバンキング開発（フロントリーダー）
     * 災害通知アプリの作成（リーダー）
     * 建築資材専門商社の顧客管理システム開発（リーダー）
   - 各プロジェクトの情報
     * 期間、役割、チーム規模
     * 業務内容の詳細
     * 成果・実績
     * 使用技術タグ

5. **Contactセクション**
   - お問い合わせフォーム
     * 名前、メールアドレス、件名、メッセージ
     * バリデーション機能
     * 送信成功メッセージ表示
   - 追加情報カード
     * 所在地（最寄駅：蕨駅）
     * 経験年数
     * 専門分野

6. **レスポンシブデザイン**
   - デスクトップ、タブレット、モバイル対応
   - ハンバーガーメニュー（モバイル）
   - フレキシブルなグリッドレイアウト

7. **インタラクティブ機能**
   - スムーズスクロール
   - スクロールアニメーション（フェードイン）
   - ナビゲーションバーのアクティブリンク表示
   - スキルバーのアニメーション
   - プロジェクトカードのチルトエフェクト
   - カーソル追従グロー効果
   - 統計カウンターアニメーション

8. **キーボードショートカット**
   - H: ホームセクションへ移動
   - A: Aboutセクションへ移動
   - S: Skillsセクションへ移動
   - P: Projectsセクションへ移動
   - C: Contactセクションへ移動

## 🛠️ カスタマイズ方法

### 個人情報の変更

**index.html**の以下のセクションを編集してください：

1. **名前と肩書き** (ヒーローセクション)
2. **自己PR** (Aboutセクション)
3. **スキル一覧** (Skillsセクション)
4. **プロジェクト実績** (Projectsセクション)
5. **連絡先情報** (Contactセクション)

### カラーテーマの変更

**css/style.css**の`:root`セクションでカラー変数を変更：

```css
:root {
    --primary-color: #00d4ff;    /* プライマリカラー */
    --secondary-color: #0088ff;   /* セカンダリカラー */
    --accent-color: #00ff88;      /* アクセントカラー */
    /* ... */
}
```

### フォントの変更

**index.html**の`<head>`セクションでGoogle Fontsのリンクを変更し、
**css/style.css**の`--font-primary`変数を更新してください。
